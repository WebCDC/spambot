# Spambot
Basic spambot created in python.

## Setup
Create python virtual environment:
```
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```
## Execute
To execute this code run it as root:
```
$ sudo bash
$ source venv/bin/activate
$ python3 spambot.py
```
After that follow the terminal instructions.
